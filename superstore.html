Superstore Performance Dashboard

  • Category: Business intelligence / Interactive data presentation
  • Technical Skills: HTML, CSS, JavaScript, data visualisation, dashboard design

Motivation: Most data analysis ends as static slides that are quickly forgotten. This project reimagines it as a browser-based experience that can be shared, opened anywhere, and explored independently without a presenter.

Overview: The dashboard explores a retail dataset across three areas: profit, discount behaviour, and order fulfilment. Each view highlights key business patterns such as margin performance, pricing impact, and delivery consistency.

How It Works: Built with HTML, CSS, and JavaScript as a single interactive report. The dashboard loads all views with optional explanations under each chart via an Elaborate button. It can be used as a live presentation or a self-guided report with toggled insights.

Impact: The format removes the need for slides or specialised tools, allowing stakeholders to explore insights at their own pace and engage directly with the data in a clear, accessible way.
